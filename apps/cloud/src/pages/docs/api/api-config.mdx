## /api/config

To download config values directly you can call `/api/config` with a `POST` request and
and an **object** with the following properties.

<Table data={meta.tableData}>
  <Table.Column prop="property" label="Property" />
  <Table.Column prop="type" label="Type" render={(i) => <code>{i}</code>} />
  <Table.Column prop="description" label="Description" />
</Table>

Invalid object requests will be returned with a zod validation error.
